# Smoke Tests

Esta carpeta queda reservada para pruebas mínimas de humo.

Casos recomendados:

- home responde `200`;
- servicios lista contenido;
- detalle de servicio responde por slug válido;
- formulario de contacto llega al endpoint;
- módulo especial responde al caso de búsqueda principal;
- healthcheck operativo en despliegue.

Regla: empezar por pruebas cortas y de alto valor antes de automatizar escenarios largos.
